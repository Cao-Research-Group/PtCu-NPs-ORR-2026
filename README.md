1. “clusters-PtCuSVac-quadElements”  file contains the clusters for Pt-Cu-OH@Pt-Vacancy CE.
2. “clusters-PtCuVac-tripleElements”  file contains the clusters for Pt-Cu-Vacancy CE.
3. "structures-trainingSet" file constains the training structures for CE (in the format of .vasp).
4. "2026-01-01-ECIs-list-PtCuOHVac-PtCuVac.xlsx" lists the fitted ECIs for both CEs. 
5. "energyList-trainingSet-PtCuSVac.txt" lists the structure names and corresponding DFT-calculated energies of Pt-Cu-OH@Pt-Vacancy training set. Elment S stands for "OH@Pt" in this project. 
6. "energyList-trainingSet-PtCuVac.txt" lists the structure names and corresponding DFT-calculated energies of Pt-Cu-Vacancy training set.
7. "PRIM-CuPtVac.vasp" is the PRIM file for Pt-Cu-Vacancy CE. 
8. "PRIM-CuPtSVac.vasp" is the PRIM file for Pt-Cu-OH@Pt-Vacancy CE.
